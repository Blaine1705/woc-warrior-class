# ⚔️ World of Claudecraft — Warrior Class

My design and implementation work on the **Warrior class** for [World of Claudecraft](https://github.com/levy-street/world-of-claudecraft), a browser-based MMORPG built by the community with AI-assisted development.

I designed and built the Warrior's three specializations end to end: their kits, combat mechanics, rage economy, and custom ability icons, shipped as pull requests to the main project.

## 🛡 Highlights

- **Three specializations** — Arms (DPS), Fury (DPS), and Protection (Tank), each with a distinct kit, signature ability, and mastery.
- **Combat stances** — Battle, Guarded, and Berserker, gated per specialization.
- **Rage economy** — dedicated generators and spenders per spec: Shieldcrack (Protection), Bloodletting and Rampage (Fury), Redhand (Arms).
- **Fury Enrage** — a short buff granting extra damage, attack speed, and movement speed, procced by Bloodletting (chance) and Rampage (always).
- **Parry** — a front-only melee avoidance stat, shown in the character sheet.
- **Specialization screen** — role, description, primary attribute, complexity, and example abilities per spec.
- **Custom ability icons** — a hand-crafted, AI-assisted icon set for the whole Warrior kit.
